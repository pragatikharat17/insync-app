# InSync 🌿
### eat in sync with your cycle

A hormone-health food app for Indian women with PCOS, PCOD, PMS and mood swings. Search 546+ Indian and Western foods with personalised scores, get AI-powered food advice in Hindi/Hinglish/English, and eat in sync with your cycle phase.

**Live app:** [insync-app.vercel.app](https://insync-app.vercel.app)
**GitHub:** [github.com/pragatikharat17/insync-app](https://github.com/pragatikharat17/insync-app)

---

## Features

- **Food search** - 546+ Indian and Western foods rated for PCOS, PCOD, PMS and mood swings (1-10), with glycemic index, inflammation score, cycle phase guide and 3 actionable hacks per food
- **AI chat ("Ask InSync")** - ask anything and get personalised answers streamed in real time, based on your condition and cycle phase
- **Cuisine filter** - switch between Indian and Western foods
- **Saved foods** - bookmark your hormone-friendly favourites
- **Meal log** - track what you eat daily
- **Editable profile** - set your name, condition (PCOS/PCOD/PMS/All) and current cycle phase - all AI answers personalise to you automatically

## How the AI works

### Data pipeline (one-time)
```
500+ Indian food names generated by Groq across 20 categories
              +
88 Western foods with GI values from published research
              ↓
For each food: structured prompt → Groq → JSON with all scores/hacks/phases
              ↓
Python import script → Supabase PostgreSQL
```

### AI chat (real-time)
```
User types question (Hindi/Hinglish/English)
              ↓
POST /api/chat (Next.js API route)
              ↓
System prompt built with user's condition + cycle phase from localStorage
              ↓
Groq API (Llama 3.3 70B) with streaming enabled
              ↓
Response streamed token by token back to UI
              ↓
User sees answer appear in real time
```

The system prompt makes the model respond like a "knowledgeable desi older sister" - warm, direct, never clinical, always specific to the user's condition and current phase.

## Food database

- **546 total foods**: ~500 Indian + ~88 Western
- **Per food data**: GI score, GI category (low/medium/high), insulin impact, inflammation score, PCOS score, PCOD score, PMS score, mood score (all 1-10), cycle phase guidance for all 4 phases, 3 actionable hacks, best food swap, best meal time, regional cooking notes
- **Cuisine tagging**: Indian vs Western for filtering
- **Indian food context**: fermentation effects, regional variations, desi cooking methods factored in
- **Western food GI**: referenced from University of Sydney GI Database

## Converting to mobile app

The app is mobile-responsive and works great in phone browsers. Three paths to native:

### Option A - PWA (recommended first step, ~30 mins)
Lets users install InSync from their browser like a native app. No App Store needed.

```bash
npm install next-pwa
```

Add `public/manifest.json`:
```json
{
  "name": "InSync",
  "short_name": "InSync",
  "description": "eat in sync with your cycle",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#E4EB9C",
  "theme_color": "#2D5128",
  "icons": [
    { "src": "/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ]
}
```

### Option B - Capacitor (fastest App Store path, 1-2 days)
Wraps the existing web app in a native shell. No rewriting needed.
```bash
npm install @capacitor/core @capacitor/cli
npx cap init InSync com.insync.app
npx cap add ios && npx cap add android
```

### Option C - Expo / React Native (2-4 weeks)
Full native app. Rewrite UI in React Native, keep Supabase + Groq backend unchanged.
